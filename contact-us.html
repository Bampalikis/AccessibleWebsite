Keyboard Navigation (Main Menu)

When the main navigation menu has focus:

  • Arrow Right / Arrow Down: Move to the next menu item.
  • Arrow Left / Arrow Up: Move to the previous menu item.
  • Home: Move to the first menu item.
  • End: Move to the last menu item.
  • Enter / Space: Activate the selected menu item.
  • Escape (when menu is open): Close the menu and return focus to the toggle button.

Use Tab and Shift + Tab to move between all focusable elements on the page.
